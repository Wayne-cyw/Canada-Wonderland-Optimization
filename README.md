## Optimization Program for Canada Wonderland with Dynamic Wait time through A* search

##### Attention:
The project should be run on terminal, and make sure to resize the terminal tab so it is big enough to show all content or it might raise an error

To run the project:

Step 1: Download the folder containing all files of this repository

Step 2: Open a new terminal tab, resize it so it is large enough, and cd to the folder

Step 3: Run main.py from the termianl

``` 
python3 main.py
``` 
```
python main.py
```
